GremlinJS Demos

    GremlinJS is intended to find and load gremlin elements automatically when they appear in the dom.

    Sample

    Add new HelloWorld gremlin

    HTML

    1. Include GremlinJS and the gremlin defintions

    <head>
    ...
    <script src="js/gremlin.min.js"></script>
    <script src="js/gremlins/HelloWorld.js"></script>
</head>

    2. Add some additional gremlin element(s) into your HTML

    Clicking the button above (see the GremlinBuilder.js for details.)

    1. creates a new div element
    2. sets the attribute data-gremlin to HelloWorld for this new element
    3. appends the element to the dom

    When the newly created element is appended to the dom, GremlinJS finds it and does what it has to.

    This happens automatically as soon as a new element with the data-gremlin attribute shows up in the dom.
