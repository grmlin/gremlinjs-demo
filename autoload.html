GremlinJS Demos

    GremlinJS is intended to find and load gremlin elements automatically when they appear in the dom.

    Sample

    HTML

    1. Include GremlinJS and the gremlin defintions

    <header>
    ...
    <script src="js/GremlinJS.min.js"></script>
    <script src="js/gremlins/HelloWorld.js"></script>
</header>

    2. Add some HTML with gremlin element(s) in it to your HTML

    Clicking the button below will

    1. create a new div-element
    2. set the attribute data-gremlin-name to HelloWorld (the gremlins used for the previous example) for this new element
    3. add the element to the dom 
        var el = document.createElement('div');
    el.setAttribute('data-gremlin-name',"HelloWorld");
    this.$pool.append(el);
