    <header>

    2. Add some HTML with gremlins in it to your HTML

    GremlinJS will detect new dom elements with a data-gremlin-name attribute, and intantiates the HelloWorld gremlin from the basic example.

    This is what happens, if the button below is clicked: 

        var el = document.createElement('div');
    el.setAttribute('data-gremlin-name',"HelloWorld");
    this.$pool.append(el);
