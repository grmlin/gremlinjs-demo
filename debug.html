GremlinJS Demos

    GremlinJS comes with a debug mode, to show you gremlins in the page and errors you made.

    Sample

    HTML

    1. Include GremlinJS and the gremlin defintions

    <head>
    ...
    <script src="js/gremlin.min.js"></script>
    <script src="js/gremlins/HelloWorld.js"></script>
</head>

    2. Add a gremlin element to your HTML

    <div data-gremlin="HelloWorld"></div>

    Javascript HelloWorld.js 

    var HelloWorld = GremlinJS.define('HelloWorld',
    function () {
        this.el.innerHTML = 'Hello World';
    }
);

    • GremlinJS.define adds a new gremlin definition. This one here is called HelloWorld.

    • The define method takes multiple arguments. In this example two of them are used:

      1. The gremlin's name

      2. The constructor function. This function is called when a gremlin with the name data-gremlin="HelloWorld" was found and instantiated.

    • this.el is available in every gremlin instance and references the gremlin's dom element.
      Here, the innerHTML property of the gremlin's dom element will be changed to display Hello World.
