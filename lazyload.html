GremlinJS Demos

    With GremlinJS it's easy to process your javascript only if it's visible.

    Think of all those fancy loading images when you scroll them into the viewport. GremlinJS does that, too, but with Javascript(s)!

    Sample

    Scroll down the site to see the gremlins appear. (best seen in the elements explorer of the browsers developer tools)

    HTML

    1. Include GremlinJS and the gremlin defintions

    <head>
    ...
    <script src="js/GremlinJS.min.js"></script>
    <script src="js/gremlins/HelloWorld.js"></script>
</head>

    2. Add a lazy gremlin element to your HTML

    <div data-gremlin-name="HelloWorld" data-gremlin-lazy="true"></div>
    • The attribute data-gremlin-lazy set to true tells GremlinJS to instantiate this gremlin only if it's in the viewport.
