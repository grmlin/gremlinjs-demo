    HTML

    1. Include GremlinJS and the gremlin defintions

    <head>
    ...
    <script src="js/GremlinJS.min.js"></script>
    <script src="js/gremlins/HelloWorld.js"></script>
    <script src="js/gremlins/Foo.js"></script>
</head>

    2. Add a gremlin element to your HTML

    <div data-gremlin-name="HelloWorld, Foo"></div>

    Javascript

    HelloWorld.js

    GremlinJS.define('HelloWorld', {
    initialize: function () {
       this.el.innerHTML = 'Hello World'
    }
});

    Foo.js

    GremlinJS.define('Foo', {
    initialize: function () {
        var p = document.createElement('p');
        p.innerHTML = 'Hello Foo?'
        this.el.appendChild(p);
    }
});

    • GremlinJS.define adds a new gremlin definition. The two gremlins here are called HelloWorld and Foo.

    • The initialize method is the constructor function called when a gremlin with the name data-gremlin-name="<NAME>" was found and initialized.

    • this.el is available in every gremlin instance and references the gremlin's dom element.
      Here, HelloWorld adds the text Hello World. Foo creates a new element with the text HelloFoo and appends it to the gremlin's dom element.
